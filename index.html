Proyectos

Heladitos

Aplicación desarrollada en Android Estudio, con el Lenguaje Java y Base de Datos NoSQL (Firebase). Esta app consiste en mostrar las ubicaciones de locales de Helado más cercanas al usuario. Este proyecto fue realizado con el propósito de ser innovador, para el ramo de estudio Proyecto de Especialidad. El cual ya fue aprobado, y estoy decidido a seguir mejorando mi aplicación, tanto en el desarrollo de interfaces y experiencia de usuario, para luego subir la app a la tienda Play Store.

Cabe destacar que la documentación de este proyecto fue realizada de forma ficticia en el tema de costos y presupuestos.
